# 📊 Customer Churn Analysis

## 🎯 Objective
Predict which telecom customers are likely to churn
and identify key drivers using machine learning.

## 🛠️ Tools Used
Python · Pandas · Scikit-learn · Matplotlib · Seaborn

## 📁 Dataset
Telco Customer Churn — 7,043 customers · 21 features
Source: Kaggle

## 🔍 Key Findings
- Overall churn rate: ~26%
- Month-to-month contracts have highest churn
- New customers (under 12 months) are highest risk
- Fiber optic users churn more than DSL users
- Model Accuracy: ~83%

## 💡 Business Recommendations
- Offer loyalty discounts to month-to-month customers
- Focus retention efforts in first 12 months
- Review pricing strategy for fiber optic plans
- Target high monthly charge customers with offers
